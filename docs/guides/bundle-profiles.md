# Bundle Profiles Directory

This directory details the **9 pre-packaged stack bundles** available via the custom installer. Use these profiles to install only what fits your project.

---

## 📦 Available Bundles

### 1. Frontend Architect (`frontend-architect`)
- **Focus**: Web UI development, framework performance, state patterns, layouts.
- **Key Skills**: `react-builder`, `nextjs-architect`, `state-management`, `tailwind-designer`, `accessibility-champion`, `pwa-builder`.

### 2. Backend & Microservices (`backend-microservices`)
- **Focus**: API design, server structures, async microservices, caching.
- **Key Skills**: `golang-engineer`, `rust-developer`, `node-fastify`, `grpc-builder`, `kafka-engineer`, `redis-cache-master`, `websocket-realtime`.

### 3. Database & Data Engineering (`database-data-ops`)
- **Focus**: Relational DB tuning, search indices, vector databases, analytics.
- **Key Skills**: `mysql-expert`, `postgresql-expert`, `mongodb-specialist`, `elasticsearch-expert`, `vector-database`, `schema-design`.

### 4. DevOps & Cloud Infrastructure (`devops-cloud-infra`)
- **Focus**: Automated pipelines, cloud services, container operations.
- **Key Skills**: `aws-architect`, `azure-developer`, `docker-expert`, `kubernetes-operator`, `terraform-builder`, `ci-cd-architect`.

### 5. AI & LLM Engineering (`ai-llm-engineering`)
- **Focus**: LLM API integrations, prompt design, RAG, vector search.
- **Key Skills**: `ai-engineer`, `llm-prompt-master`, `rag-builder`, `ai-safety-guard`, `langchain-expert`.

### 6. Mobile Engineering (`mobile-engineer`)
- **Focus**: Flutter, React Native, native platforms, push, biometric, store builds.
- **Key Skills**: `flutter-architect`, `react-native-builder`, `app-store-optimizer`, `biometric-auth`, `mobile-ci-cd`.

### 7. Security & Pentesting (`security-pentester`)
- **Focus**: OWASP defenses, security auditing, container/cloud hardening.
- **Key Skills**: `security-auditor`, `owasp-guard`, `pen-tester`, `jwt-security`, `zero-trust-architect`.

### 8. PHP Frameworks & CMS (`php-cms-suite`)
- **Focus**: Laravel stacks, CodeIgniter, CakePHP, WordPress Plugins, Themes.
- **Key Skills**: `laravel-architect`, `laravel-nova`, `codeigniter-rest`, `wordpress-plugin`, `cakephp-developer`.

### 9. Developer Experience & QA (`dx-qa-automation`)
- **Focus**: Unit tests, Playwright automations, conventions, changelogs.
- **Key Skills**: `unit-test-writer`, `e2e-automator`, `bdd-cucumber`, `linting-configurator`, `commit-convention`.
